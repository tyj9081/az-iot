## API 调用约定

```typescript
// src/api/xxx.ts — 所有 API 必须经此层
import request from '@/utils/request'
export const xxxApi = {
  list: (params) => request.get('/xxx', { params }),
  getById: (id) => request.get('/xxx/' + id),
  create: (data) => request.post('/xxx', data),
  update: (id, data) => request.put('/xxx/' + id, data),
  remove: (id) => request.delete('/xxx/' + id)
}
```

- **禁止** 在 .vue 文件中直接 `axios.get('/api/v1/xxx')`
- 请求拦截器自动注入 `Authorization: Bearer {token}`
- 401 响应自动清除 token 跳转登录页

## 通用渲染器

设备详情页通过点表 `data_type` 驱动渲染：

```vue
<template v-for="item in readings">
  <!-- Numeric → ValueCard -->
  <div v-if="isNumeric(item.dataType)" class="reading-card">
    <p class="label">{{ item.sensorName }}</p>
    <p class="value">{{ item.value }} <span class="unit">{{ item.unit }}</span></p>
  </div>
  <!-- Bool → StatusLight -->
  <div v-else-if="item.dataType === 'bool'" class="reading-card">
    <div :class="['dot', item.value > 0 ? 'on' : 'off']"></div>
    <p>{{ item.sensorName }}</p>
  </div>
</template>
```
