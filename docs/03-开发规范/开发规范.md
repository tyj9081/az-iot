# 开发规范

> AZ-IOT S01-S07 | 强制执行，AI 必读

---

## S01 — API 契约

- 统一响应 `{code:0, message:"success", data:..., timestamp}` (code: 0/400/401/403/404/500)
- 分页 `{records, total, page, pageSize}`, page 从 1 开始
- JSON camelCase, 日期 `yyyy-MM-dd HH:mm:ss`
- **强制**: DTO 强类型，禁止 `Map<String,Object>` 做入参
- Controller: `@RequestMapping("/api/v1/xxx")`

---

## S02 — 数据库

- 表前缀: 业务 `dev_`, 系统 `sys_`
- 主键 `id BIGINT AUTO_INCREMENT`
- 审计字段: `created_at`, `updated_at ON UPDATE CURRENT_TIMESTAMP`
- **禁止** ENUM (用 VARCHAR); **禁止** 关键字段 NULL
- 外键字段必有索引, 命名 `idx_{表简写}_{列名}`
- Flyway: `V{序号}__{描述}.sql`, **禁止修改已执行文件**
- 逻辑删除: `is_deleted TINYINT DEFAULT 0` + `@TableLogic`

---

## S03 — MQTT

- Topic: `neuron/{collector_mqtt_client_id}/{type}`
- type: reading, latest, heartbeat, alarm, config/delta, config/full, config/ack, cmd, cmd/result
- 荷载 JSON, **禁止包含 Java 包名或 Rust 模块路径**
- 通道A (latest): QoS 0, fire-and-forget
- 通道B (reading): QoS 1, 批量 ≤500 条
- quality: 0=正常, 1=超时, 2=异常, 3=传感器故障

---

## S04 — Java 代码

- 构造器注入 `@RequiredArgsConstructor`, **禁止 @Autowired 字段注入**
- Controller→Service→Mapper 分层, **禁止 Controller 直接调 Mapper**
- Service 返回 VO, **禁止 Controller 暴露 Entity**
- 包结构按业务域: `entity.device/collector/system`
- 全局异常处理 `@RestControllerAdvice`, **禁止 Controller 内 try-catch 吞异常**
- 日志 `@Slf4j`, 禁止 `e.printStackTrace()`

---

## S05 — Rust 代码

- `cargo clippy` 零警告, `cargo fmt` 统一格式
- **禁止 `unwrap()`** 非测试代码 (用 `?` 或 `expect`)
- **禁止吞错误** `let _ = ...`; **禁止 `println!`** 用 `tracing`
- Protocol trait: `async fn collect`, 内部 `spawn_blocking` 包裹同步 IO
- 错误处理: `anyhow::Result` 顶层 + `thiserror` 结构化

---

## S06 — Vue 3 代码

- `<script setup lang="ts">`, Composition API, Composables 复用
- Props/Emits 声明类型, **禁止 `any`** (用 `unknown`+类型守卫)
- 组件命名: PascalCase, views 目录 kebab-case
- API 调用经 `src/api/` 封装, **禁止 .vue 内直接 axios**
- 状态: 全局→Pinia, 页面级→ref/reactive
- TypeScript strict mode

---

## S07 — 测试规范

- **铁律: 业务代码严禁任何 mock, 所有测试走真实实现**
- Java: `@SpringBootTest + H2`, `@Transactional` 回滚
- Rust: 已知字节→已知数值, 每种 data_type × 4 byte_order 全覆盖
- 提交前置: `mvn verify` / `cargo test && cargo clippy` / `pnpm build`
- **禁止** `mock(Service.class)` / `when().thenReturn()` / 空壳测试
