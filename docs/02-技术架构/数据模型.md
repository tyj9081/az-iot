## 实体关系

```
dev_protocol ──┐
               │ 1:N
dev_manufacturer ──1:N──→ dev_device_model ──1:N──→ dev_register_map
                                     │
                                     │ 1:N
                                     v
                               dev_device ──1:N──→ dev_device_reading
                                     │
                                     │ N:1
dev_collector ──1:N──→ dev_serial_port ──┘

sys: user, role, permission, user_role, role_permission, audit_log, config
```

---

## 系统表 (sys_)

### sys_user
用户表，密码 BCrypt 哈希存储，初始管理员 admin/admin123

### sys_role
角色表，预置 admin / operator / viewer

### sys_permission
权限表，树形结构 (parent_id)，支持 menu/button/api 三种类型

### sys_user_role
用户-角色 N:M 关联

### sys_role_permission
角色-权限 N:M 关联

### sys_audit_log
审计日志，通过 @AuditLog AOP 切面自动记录

### sys_config
键值对系统配置，如 data_retention_days = 90

## 设备表 (dev_)

### dev_protocol
协议字典，预置: ModbusRTU, ModbusTCP, DL_T645, BACnet_IP, SNMP, ATCommand

### dev_manufacturer
品牌/厂家，字段: code, name, country, website

### dev_device_model
设备型号，关联 manufacturer_id + protocol_id，含默认采集间隔 900s

### dev_register_map
寄存器点表，关联 model_id (CASCADE 删除)，字段: sensor_code, register_address, data_type, byte_order, func_code, coefficient, offset_val, unit, rw
- 唯一约束: (model_id, sensor_code)

### dev_collector
采集器 (BC-U101)，字段: mqtt_client_id (唯一，作为 MQTT topic 标识)

### dev_serial_port
串口配置，关联 collector_id (CASCADE)，bus_param 存 JSON
- 唯一约束: (collector_id, port_name)
- BC-U101 默认: COM1=sms_modem, COM2=io_board, COM3-10=device

### dev_device
终端设备，关联 serial_port_id + model_id，slave_addr 为 Modbus/645 从站地址
- collect_interval_sec NULL = 继承型号默认

### dev_device_reading **[分区表]**
时序读数，按月 RANGE 分区 (TO_DAYS(read_at))
- 关键字段: device_id, register_id (FK), sensor_code(冗余), value, avg, max, min, sample_count
- 索引: idx_device_time, idx_register_time, idx_device_register
- 分区: p202606, p202607, ..., p_future (MAXVALUE)
