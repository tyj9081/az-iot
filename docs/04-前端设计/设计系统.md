## 组件库 (Element Plus)

- el-table — 数据表格, 带分页器
- el-dialog — 新增/编辑弹窗
- el-select — 下拉选择 (级联联动)
- el-input / el-input-number — 输入框
- el-tag — 状态标签 (在线=绿, 离线=灰, 告警=红)
- el-button — 操作按钮, type=primary #534AB7
- el-form — 表单验证

---

## 通用渲染器

同一组件根据 `data_type` 自动选渲染:

| data_type | 组件 | 样式 |
|-----------|------|------|
| float32/64, int16/32, uint16/32 | 数值卡片 | 24px 数值 + 13px 单位 + 右上角"趋势" |
| bool | 状态灯 | 28px 圆点(绿/红/灰) + 标签文字 |
| bool + rw=RW | 灯 + 开关 | 同 bool + el-switch 控制 |
